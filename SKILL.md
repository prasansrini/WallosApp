# WallosApp Development Guidelines

This repository enforces the use of the `caveman` and `ponytail` skills for all agent operations.

## 🛠️ Required Skills

When working on this repository, you **MUST** adhere to the following guidelines:

1. **[ponytail](file:///C:/Users/Phoex/.gemini/config/plugins/ponytail/skills/ponytail/SKILL.md)**: 
   * Write only what is strictly necessary. Never over-engineer.
   * Prioritize native browser/platform features, standard libraries, and existing dependencies over writing new custom code.
   * Run the `/ponytail` command (or activate `ponytail` mode) to analyze and trim code down to its minimal, cleanest working state.

2. **[caveman](file:///C:/Users/Phoex/.gemini/config/plugins/caveman/skills/caveman/SKILL.md)**:
   * Keep prose responses extremely concise and to the point.
   * Adopt a terse conversational style to minimize token consumption and speed up iterations.
